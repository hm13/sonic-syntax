# 構文S
S := t | t -> S
t := t_45-34 | t_34-23 | t_23-34 | t_34-45
t_45-34 := 4-sonic
t_34_23 := 3-sonic
t_23-34 := 3-sonic-rev
t_34-45 := 4-sonic-rev

# 判定J(e)
J(e) := e ~>* ε
  t ~> ε
  t_*-mn -> t_mn-*       ~> ε
  t_*-mn -> t_mn-* -> e' ~> e'

# 判定J(e)の性質
指の間のペンの遷移が連続的となる式eを実行可能と呼ぶことにすると次が成り立つ。
(性質1) 実行可能 => 判定J(e)が真
∵ ペンの遷移が連続的であるため、隣り合う全ての技tの終了指番号と次の技t'の開始指番号は等しい。必ず規則が適用できるためJ(e)は真。
(性質2) 実行可能 <= 判定J(e)が真
∵ eの隣り合う全ての技tの終了指番号と次の技t'の開始指番号が等しい。これは実行可能。
